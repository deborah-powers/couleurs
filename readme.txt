ma première progressive web app
c'est un nuancier, qui permet de sélectionner et comparer deux couleurs html.

les sections:
	une barre de texte
	deux blocs colorés
	un nuancier de couleurs codées
	un nuancier de couleurs nommées

les blocs
	cliquer sur un bloc
	cliquer sur une case colorée d'un nuancier
	le bloc change de couleur
	il est possible de comparer deux couleurs

la barre de texte
	entrer le nom d'une couleur
	le bloc de gauche se colore
